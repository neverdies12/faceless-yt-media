Privacy Policy

Last updated: 2026

This app ("Faceless Facts Poster") is a personal automation tool built and operated by a single individual to publish original short-form educational video content to their own social media accounts (YouTube, TikTok, Instagram).

What this app does

The app generates original video content and uploads it directly to the developer's own connected accounts on the platforms above, using each platform's official API. It does not operate as a public-facing service, and no third party interacts with it.

Data collected

The app requests only the minimum account-identification scope needed to authenticate and post to the developer's own connected account (for example, TikTok's user.info.basic scope, used solely to confirm which account is authorized for posting). No personal data belonging to any other user is collected, processed, or stored.

Data sharing

No data is sold, shared, or disclosed to any third party. Authentication tokens are stored locally and used only to make authorized API calls to publish content to the developer's own accounts.

Data retention

Authentication tokens are retained only for as long as needed to operate the posting automation and can be revoked at any time by disconnecting the app from the relevant platform's account settings.

Contact

Questions about this policy can be directed to the developer via the contact information on file with each platform's developer portal.
